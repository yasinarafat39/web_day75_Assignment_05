What is the Document Object Model (DOM)?

DOM or Document Object Model is a programming interface of XML or HTML document. DOM represant our HTML or XML document so that we can easily modify the page with the programming language.


We can do with DOM look like these:

  • Create new elements.
  • Remove the existing element.
  • Update the existing element.
  • Attributes Update, Add, Remove.
  • Add, Update, and Remove the Style.
  • Create a event in specific a element.
How do you select an element from the DOM?

We can select an element from the DOM with element selectors:

  • ID selector:
    • document.querySelector('#idName')
    • document.getElementById(idName)
  • Class selector:
    • document.querySelector('.className')
    • document.querySelectorAll('.className')
    • document.getElementByClassName(className)
  • Tag Name selector:
    • document.querySelector('div')
    • document.querySelectorAll('h4, section, article')
    • document.getElementByTagName(div)
  • Attributes selector:
    • document.querySelector('[id="idName"]')
    • document.querySelectorAll('[id="idName"], [id="idName"]')
  • Pseudo-Class selector:
    • document.querySelector('li:first-child')
    • document.querySelectorAll('li:first-child, li:last-child')
  • Children selector:
    • const parentElement = document.getElementByTagName(div)
    • parentElement.children
    • parentElement.childNodes
  • Parent selector:
    • const anElement = document.getElementByTagName(div)
    • anElement.parentNode
    • anElement.parentNode.parentNode
  • Without specific element selector:
    • document.querySelectorAll('.box:not(#box3)')
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element to handle events triggered by its child elements. Instead of attaching event listeners to each individual child, the event is "delegated" to the parent. This works because events in the DOM "bubble up" from the target element (where the event occurs) to its ancestors (including the parent), allowing the parent to catch the event.


  • How it works:
    • An event listener is attached to a parent element.
    • When an event occurs on a child element, it bubbles up through the DOM tree.
    • The parent element's event listener detects the event and determines whether the event originated from a specific child.

  • Benefits of event delegation:
    • Efficiency: Fewer event listeners need to be attached, reducing memory consumption, especially with a large number of child elements.
    • Dynamic content handling: It works for child elements that are dynamically added to the parent later on. There's no need to reattach event listeners for new elements.
    • Maintainability: Centralizing event handling in a single function makes code easier to manage and debug.

  • Use cases:
    • Handling events on lists, tables, or grids.
    • Dealing with dynamic content where elements are added or removed frequently.
How do you manipulate an element's attributes and styles using the DOM?

Manipulating an element's attributes and styles using the DOM can be done through various methods. Hereâ€™s how we can do it:


  • Manipulating Attributes:

    You can use methods and properties like getAttribute(), setAttribute(), removeAttribute(), and direct property access to manipulate attributes.

    • Getting an Attribute:
      • const link = document.querySelector('a')
      • const href = link.getAttribute('href')
    • Setting an Attribute:
      • link.setAttribute('href', 'https://google.com')
    • Removing an Attribute:
      • link.removeAttribute('href')
  • Manipulating Styles:

    To manipulate an element's CSS styles, we can either modify individual properties using the style property or apply/remove CSS classes.

    • Using the style property:
      • const div = document.querySelector('div')
      • div.style.backgroundColor = 'blue';
      • div.style.width = '200px';
    • Using CSS Classes:
      • Adding a class:
      • div.classList.add('highlight');
      • Removing a class:
      • div.classList.remove('highlight');
    • Using the className:
      • const div = document.querySelector('div')
      • div.className = 'flex items-center gap-4';
